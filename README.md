# 🚀 Git Easy Commit

Um **CLI simples para fazer commits mais rápidos e organizados** no Git usando **Conventional Commits + emojis**.

A ideia é **não precisar lembrar comandos do Git nem padrões de commit**.  
Você apenas executa um comando, escolhe o tipo de commit e escreve a descrição.

O script executa automaticamente:

- `git add .`
- `git commit`
- `git push`

Tudo de forma **interativa no terminal**.

---

# ✨ Exemplo de Uso

Execute no terminal:

```bash
gcommit
```
